<!-- From dropbox_handler.py — extract_episode_number regex patterns to reuse -->
From dropbox_handler.py:
```python
def extract_episode_number(self, filename):
    patterns = [
        r"[Ee]pisode\s*(\d+)",  # Episode 25
        r"[Ee]p[_\s]*(\d+)",    # Ep 25, ep_25, ep25
        r"^(\d+)\s*[-_]",       # 25 - Title
        r"#(\d+)",              # #25
    ]
```
This must be extracted into a module-level function in rss_episode_fetcher.py so both RSS and local-file ingest paths can use it without importing DropboxHandler.
