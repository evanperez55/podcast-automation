       - DOCTYPE html with lang="en"
       - head: charset, viewport, title="{{ episode_title }}"
       - head: JSON-LD script block with PodcastEpisode structured data (name, description, url, datePublished, episodeNumber, inLanguage, partOfSeries with PodcastSeries)
       - head: Open Graph meta tags (og:type="article", og:title, og:description, og:url, og:site_name="Fake Problems Podcast", og:image if thumbnail_url)

       - body: h1 with episode title

       - body: div.transcript with segments, each as span id="t-{start_seconds}" class="segment"
       - Use Jinja2 autoescaping (autoescape=True in Environment) for XSS protection
       - Basic CSS in style block: readable font, max-width 800px centered, chapter nav as horizontal/vertical list

       - Class `EpisodeWebpageGenerator` with `self.enabled = os.getenv("PAGES_ENABLED", "true").lower() == "true"`
       - Env vars: GITHUB_TOKEN, GITHUB_PAGES_REPO, SITE_BASE_URL, GITHUB_PAGES_BRANCH (default "main")

       - `generate_html(episode_number, analysis, transcript_data, audio_url=None, thumbnail_url=None) -> str`: Load Jinja2 template, build context dict with episode_title, episode_summary, episode_url, datePublished, chapters, transcript segments, keywords, JSON-LD data. Return rendered HTML string.
       - `generate_sitemap(existing_xml: str | None, new_url: str) -> str`: Parse existing sitemap XML (if any), extract existing loc URLs, add new_url if not already present, return complete sitemap XML string. Use xml.etree.ElementTree.
       - `_build_episode_url(episode_number: int) -> str`: Construct URL from SITE_BASE_URL + f"/episodes/ep{episode_number}.html"
       - Jinja2 Environment: loader=FileSystemLoader pointing to templates/ dir relative to module file, autoescape=True
