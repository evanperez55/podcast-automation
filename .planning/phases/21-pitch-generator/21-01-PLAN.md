    - generate_demo_pitch(client_slug, episode_id): load YAML + DEMO.md + analysis JSON, build demo system prompt (same style but reference specific metrics — LUFS delta, clip count, episode title, show note excerpt), call GPT-4o, parse response, write PITCH.md to demo/<slug>/<ep_id>/PITCH.md, return dict
    - _load_prospect_yaml(client_slug): read clients/<slug>.yaml, return podcast_name + prospect block
    - _load_demo_md(client_slug, episode_id): read demo/<slug>/<ep_id>/DEMO.md as text
    - _load_analysis(client_slug, episode_id): glob *_analysis.json in output/<ep_id>/, sort by mtime, return newest parsed
    - _call_openai_with_retry(system_prompt, user_message, max_retries=3): exact pattern from content_editor.py (exponential backoff on RateLimitError, APIError, APIConnectionError, APITimeoutError)
    - _parse_pitch_response(raw): split on "### SUBJECT", "### EMAIL", "### DM" markers, return dict
    - _write_pitch_md(path, pitch): write markdown with subject/email/dm sections, mkdir parents
    - run_gen_pitch_cli(argv): standalone function for CLI dispatch (keeps main.py thin). Parse argv: gen-pitch <slug> [ep_id]. If no slug, print usage. If ep_id present, call generate_demo_pitch. Else call generate_intro_pitch. Print result path or error.
