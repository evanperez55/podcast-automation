Signature:
```python
def run_demo_workflow(
    slug: str,
    episode_id: str,
    args: dict,
    consent_fn=None,
) -> Optional[Path]:
```

`consent_fn` defaults to `None`. When `None`, use `input()` to prompt: `"CONSENT CHECK: Have you obtained explicit permission from the host of '{show_name}' to process their episode? Type 'yes' to confirm: "`. When provided (for testing), call `consent_fn(prompt_text)` instead.

Flow:
1. Create `OutreachTracker()`, call `get_prospect(slug)` — raise `ValueError(f"Prospect '{slug}' not found in tracker")` if None
2. Show prospect info (show_name, status, contact_email if present)
3. Call `consent_fn` (or `input()`). If response.strip().lower() != "yes", print "Aborted — no consent confirmed." and return None
4. Update tracker status to "interested" (the prospect has agreed to a demo)
5. Activate client config: `from client_config import activate_client; activate_client(slug)`
6. Run pipeline: `from pipeline import run_with_notification; run_with_notification(args, episode_number=int(episode_id.replace("ep", "")))` — extract episode number from ep_id string
7. Package demo: `DemoPackager().package_demo(slug, episode_id)`
8. Generate pitch (try/except, log warning on failure): `from pitch_generator import PitchGenerator; PitchGenerator().generate_demo_pitch(slug, episode_id)`
9. Update tracker status to "demo_sent"
10. Print summary: demo folder path, pitch path (if generated), next steps ("Review the demo output, then send the pitch manually")
11. Return the demo folder Path

Also add `run_demo_workflow_cli(argv: list, args: dict) -> None` below `run_demo_workflow`:
```python
def run_demo_workflow_cli(argv, args):
    slug = argv[2] if len(argv) > 2 else None
    ep_id = argv[3] if len(argv) > 3 else None
    if not slug or not ep_id:
        print("Usage: uv run main.py demo-workflow <slug> <ep_id>")
        return
    try:
        run_demo_workflow(slug, ep_id, args)
    except ValueError as e:
        print(f"Error: {e}")
    except Exception as e:
        print(f"Error during demo workflow: {e}")
```
