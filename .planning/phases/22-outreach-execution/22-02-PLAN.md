  <name>Task 1: Execute real prospect outreach using the demo-workflow command</name>
  <action>
This task requires the user to perform real-world outreach. Claude cannot do this.

**Steps for the user:**

1. **Find a prospect** (if not already identified):
   - Run `uv run main.py find-prospects --genre <genre> --min-episodes 20 --max-episodes 500`
   - Or use a prospect already in the tracker: `uv run main.py outreach list`

2. **Get consent** from the prospect:
   - Send an intro pitch first: `uv run main.py gen-pitch <slug>`
   - DM or email the prospect asking permission to process one of their episodes as a free demo
   - Wait for their explicit "yes" before proceeding

3. **Run the demo workflow**:
   ```bash
   uv run main.py --client <slug> demo-workflow <slug> <ep_id>
   ```
   - Type "yes" at the consent prompt to confirm you have permission
   - Wait for pipeline processing (~40 min on GPU)

4. **Review the output** before sending:
   - Check `demo/<slug>/<ep_id>/` folder contents
   - Read the blog post and social captions — do they match the show's voice?
   - Read the PITCH.md — does it reference specific episode details?
   - If voice persona is off, update `clients/<slug>.yaml` and re-run

5. **Send the pitch**:
   - Copy the email content from `demo/<slug>/<ep_id>/PITCH.md`
   - Send manually from your personal email
   - Update tracker: `uv run main.py outreach update <slug> demo_sent`

6. **Verify tracker state**:
   - Run `uv run main.py outreach list` — confirm prospect shows at "demo_sent" status
   - Run `uv run main.py outreach status <slug>` — confirm last_contact_date is today
