## Prospect Qualification Checklist

Before processing a sermon, verify:

- [ ] Has weekly sermon podcast or YouTube uploads
- [ ] No YouTube Shorts (or very few)
- [ ] No sermon transcripts on website
- [ ] No blog posts derived from sermons
- [ ] Social media exists but doesn't repurpose sermon content
- [ ] Church size appears to be 200-2,000 (check About page, staff size, building photos)
- [ ] Contact info found (pastor email, comms director, or general contact)
- [ ] RSS feed URL confirmed (for pipeline processing)

## Processing a Prospect

```bash
# 1. Create client config
cp clients/church-template.yaml clients/{slug}.yaml
# Edit: set podcast_name, rss_source.feed_url, client_name

# 2. Process latest sermon
uv run main.py --client {slug} latest --auto-approve

# 3. Copy PITCH_TEMPLATE.md, fill in specifics
cp demo/church-vertical/PITCH_TEMPLATE.md demo/church-vertical/{slug}/PITCH.md

# 4. Upload clips to Google Drive, fill in Drive link

# 5. Send pitch email
```
